# Funktionen definieren in Haskell

*Hinweise: Ein Funktionsname und alle Eingabeparameter beginnen in Haskell immer mit einem Kleinbuchstaben.*

## Aufgabe 1: Haskell – Eine mystische Funktion

a) Berechnen Sie, was folgende Funktion liefert, und beschreiben Sie kurz (1-2 Sätze) den Aufbau dieser Funktion.

```haskell
mystic :: Int -> Int -> Int
mystic a b = div a 2 +(3*a - b)
```

b) Mit welchem Haskell-Befehl kann man die Signatur einer Funktion ermitteln lassen. Testen Sie dies für die Funktion mystic.
